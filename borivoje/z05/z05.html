Šta je HTML?

  • HTML je jezik za opisivanje web strana
  • HTML - Hyper Text Markup Language
  • Markup language je skup markup tagova
  1. HTML tagovi
    1. Tag je ključna reč okružena sa < i >, npr
    2. Tagovi su najčešće u paru,
    3. Prvi je start tag (opening tag), a drugi end tag (closing tag)
    4. Ne razlikuju se velika i mala slova – preporuka je mala
  2. HTML dokument = Web strana
    1. Web browser čita HTML dokument i prikazuje ga kao web stranu
    2. Ime HTML dokumenta se završava sa .htm ili .html
    3. Za pregled u browser-u RUN -> LAUNCH IN CHROME
  3. HTML naslovi (Headings)
    1. Koriste se tagovi h1 do h6
    2. Važnost pravilne upotrebe naslova
  4. HTML pasusi (Paragraphs) / prelomi reda (Line Breaks)
    • Koriste se tagovi p i br
  5. HTML VEZE (Links)
    • Koristi se tag a
    • Veza može pokazivati na bilo koji resurs na netu - HTML stranu, sliku, zvičnu ili video datoteku...
    • TARGET atribut određuje gde se otvara link
    • NAME atribut određuje imenovano sidro u HTML dokumentu
  6. SLANJE mail-a
    7. href="mailto: ime@domen.com ?Subject=PredmetPoruke"
  7. HTML SLIKE (Images)
    8. Koristi se tag img
    Npr: img src="/slika.jpg" /
    Alt atribut određuje alternativni tekst za sliku
    Npr:img src="/slika.gif" alt="Opis slike" /
    Podešavanje širine / visine - Atributi width i height
  8. HTML ATRIBUTI (Attributes)
    • Atributi obezbeđuju dodatne informacije o elementu
    • Uvek se navode u start tagu
    • Pišu se u obliku imeatributa="vrednostatributa"
  9. HTML LINIJE (Lines)
    • Koristi se tag hr/
  10. HTML FORMATIRANJE TEKSTA
    11. Koriste se tagovi za formatiranje (Formatting Tags):
    • b - podebljani (bold) tekst
    • big - veća slova
    • em - naglašen tekst
    • i - kurziv (italic) tekst
    • small - manja slova
    • strong- jača sloba, podebljani tekst
    • sub - indeks
    • sup - eksponent
    • ins - umetnuti tekst
    • del - obrisani tekst
    • code - računarski kod
    • abbr - skraćenica
    • cite- citat
  11. HTML STILOVI (Styles)
    12. Stilovi obezbeđuju univerzalan način za doterivanje HTML elemenata
    Mogu se primeniti direktno na HTML element korišćenjem style atributa ili indirektno pomoću posebne CSS datoteke
    Primeri:
    • Boja pozadine - body style="background-color:yellow"
    • Font, boja, pozadina - p style="font-family:courier new; color:red; font-size:20px"
    • Poravnanje - h1 style="text-align:center"
  12. HTML TABELE (Tables)
    13. Koriste se sledeći tagovi:
    • table - za tabelu
    • tr - za red
    • td - za ćeliju
    • th - za ćelije u zaglavlju
    • caption - za naslov (odmah iz table taga)
    Sadržaj ćelije može biti tekst, slike, liste, pasusi, forme, horizontalne linije, tabele...
    Atributi:
    • border - određuje okvir
    • colspan - spaja kolone
    • rowspan - spaja redove
    • cellpadding - dodaje prostor oko sadržaja ćelija
    • cellspacing - dodaje prostor između ćelija
    • bgcolor – postavlja boju pozadine
    • background – postavlja sliku kao pozadinu
    • width/height – postavljaju širinu/visinu u px ili %
    • align – određuje poravnanje sadržaja
    • Da bi tabela očuvala pravilnu strukturu u prazne ćelije treba staviti
  13. HTML LISTE (Lists)
    1. Nenumerisana lista (Unordered List)
      • Počinje ul tagom
      • Svaka stavka počinje li tagom
      • Atribut type:
        1. square - kvadrat
        2. circle - krug
        3. disc - disk
    2. Numerisana liste (Ordered List)
      • Počinje ol tagom
      • Svaka stavka počinje li tagom
      • Atribut type:
        1. A - velika slova
        2. a - mala slova
        3. I - rimski brojevi
        4. i - mali rimski brojvi
        5. 1 - normalna numeracija (podrazumevano)
    3. Lista definicija (Definition List)
      • dl - lista definicija
      • dt - pojam
      • dd - definicija pojma
    4. Ugnježdene liste
